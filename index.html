Erdős formal-proof fidelity audit

Which “formally solved” Erdős problems rest on an unconditional Lean proof — and which silently assume an unproven result.

A formal proof can be sorry-free and #print axioms-clean and still prove the problem only conditionally — by taking a deep theorem as a hypothesis parameter, which the axiom check cannot see. This reads each hosted Lean proof mechanically and reports it, then cross-references the (now frozen) AI-contributions wiki to surface where a problem recorded as solved rests on a proof that is not. No human or model judgment in the machine layer; signed verdicts carry a named reviewer.
